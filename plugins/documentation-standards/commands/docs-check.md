Run a comprehensive documentation quality check on all markdown files in the project.

This command will:

1. Find all markdown files
2. Run rumdl to check markdown formatting
3. Run vale to check prose quality
4. Check zensical compatibility if applicable
5. Report all issues found

Use this command before committing documentation changes to ensure they meet standards.
