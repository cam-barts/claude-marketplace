Use MegaLinter to discover quality issues and recommend appropriate tools.

This command will:

1. Run MegaLinter analysis on repository
2. Generate comprehensive quality report
3. Identify high-value linters with many findings
4. Recommend specific tools for prek configuration
5. Suggest GitHub Actions integration
6. Prioritize tools with autofix capabilities

Use to discover what quality tools would be most valuable for the project.
