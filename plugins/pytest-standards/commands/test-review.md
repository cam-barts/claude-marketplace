Review existing tests for adherence to pytest best practices.

This command will check for:

- Proper test naming conventions
- Appropriate fixture usage (not over/under-used)
- Mocking with autospec
- Opportunities for property-based testing with Hypothesis
- Parametrization opportunities
- Test organization and structure
- Assertion quality (outcomes vs steps)

Provides actionable recommendations for improving test quality and maintainability.
